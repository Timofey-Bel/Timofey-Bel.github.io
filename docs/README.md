## 🆘 Нужна помощь?

1. Проверьте раздел "Решение проблем" в соответствующем документе
2. Убедитесь, что используете PHP 8.5: `php -v`
3. Проверьте логи: `../storage/logs/laravel.log`
4. Вернитесь к [DOCS_INDEX.md](DOCS_INDEX.md) для полной навигации
