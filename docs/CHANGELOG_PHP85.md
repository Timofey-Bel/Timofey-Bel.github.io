### Требования

Теперь проект требует:
- **PHP 8.5 или выше** (обязательно)
- Laravel 12.x
- Composer (последняя версия)

### Инструкции по обновлению

1. **Установите PHP 8.5 в OSPanel:**
   - Откройте OSPanel → Настройки → Модули
   - Выберите PHP 8.5
   - Перезапустите OSPanel

2. **Проверьте версию:**
   ```bash
   php -v
   ```

3. **Обновите зависимости:**
   ```bash
   composer update
   ```

После обновления проверьте:

```bash

# Список маршрутов
php artisan route:list

# Запуск сервера
```

### Возможные проблемы

#### Composer не может найти совместимые пакеты
**Решение:**
```bash
composer update --with-all-dependencies
```

#### OSPanel не видит PHP 8.5
**Решение:**
1. Скачайте PHP 8.5 для Windows
2. Распакуйте в `C:\OSPanel\modules\PHP-8.5\`
3. Перезапустите OSPanel
